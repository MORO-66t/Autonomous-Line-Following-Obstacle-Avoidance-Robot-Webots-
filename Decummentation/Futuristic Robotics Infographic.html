PID LINE FOLLOWING

Error Calculation

Error Formula
error = right_sensor - left_sensor
Positive error: Robot drifting left
Negative error: Robot drifting right
Zero error: Robot centered on line

PID Correction

Correction Formula
correction = Kp × error + Kd × derivative
Proportional
Kp = 0.005
Present error
Derivative
Kd = 0.001
Rate of change

Differential Drive Steering

Left Motor: base_speed + correction
Right Motor: base_speed - correction

Creates smooth turning motion to correct trajectory

Scenario: Centered on Line

🤖 Centered
Error = 0 • No correction needed

Scenario: Drifting Left

🤖
Error > 0 • Steer right to correct

Scenario: Drifting Right

🤖
Error < 0 • Steer left to correct

Controller Architecture

📥Input Processing

gs_values = read ground sensors
ps_values = read distance sensors
obstacles = detect thresholds

🧮State Logic

if obstacle_front:
state = STOP
elif line_detected:
state = FOLLOW_LINE

⚙️PID Calculation

error = right_gs - left_gs
derivative = error - last_error
correction = Kp × error + Kd × derivative

📤Motor Output

left_speed = base + correction
right_speed = base - correction
motors.setVelocity(speeds)
Control Loop Execution

The main control loop executes at simulation timestep rate (typically 64ms), continuously reading sensors, updating state machine, calculating PID corrections, and commanding motors for real-time autonomous control.
